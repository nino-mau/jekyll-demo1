# GitHub-Style Alerts Demo

This page demonstrates all available GitHub-style alert types.

## Note (Blue - Information)

<div class="alert alert-note">
<p><strong>Note</strong></p>
<p>Useful information that users should know, even when skimming content.</p>
</div>

## Tip (Green - Helpful Advice)

<div class="alert alert-tip">
<p><strong>Tip</strong></p>
<p>Helpful advice for doing things better or more easily.</p>
</div>

## Important (Purple - Key Information)

<div class="alert alert-important">
<p><strong>Important</strong></p>
<p>Key information users need to know to achieve their goal.</p>
</div>

## Warning (Orange - Urgent Attention)

<div class="alert alert-warning">
<p><strong>Warning</strong></p>
<p>Urgent info that needs immediate user attention to avoid problems.</p>
</div>

## Caution (Red - Risks)

<div class="alert alert-caution">
<p><strong>Caution</strong></p>
<p>Advises about risks or negative outcomes of certain actions.</p>
</div>

## Multiple Paragraphs

<div class="alert alert-note">
<p><strong>Note</strong></p>
<p>You can include multiple paragraphs in an alert box.</p>
<p>This is the second paragraph with additional information.</p>
<p>And even a third paragraph if needed!</p>
</div>

## With Code Examples

<div class="alert alert-tip">
<p><strong>Tip</strong></p>
<p>You can use inline code like <code>jekyll serve</code> within alerts.</p>
<p>Or reference configuration files: <code>_config.yml</code></p>
</div>
